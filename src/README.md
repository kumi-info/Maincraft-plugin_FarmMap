# FarmMap（農場マップ）v1.3.0

s2e-farm-pro のアリーナ（フロア）設定を読み取り、その周囲に **横に広がる農場マップ** を
バッチ設置で生成する補助プラグイン。高さのある“壁”は作らず、装飾余白(margin)帯へ平らに
農場（建物・背景）が広がる。現在のデザインスタイル = **#1「木の牧場」**（他スタイルは今後追加）。

## できること（v1.1.0）

- **平らな土台（横に広がる）**: 内側プレイフィールドは耕地 `FARMLAND`（作物は床の上に育つので非干渉）、外側の余白帯は `GRASS_BLOCK`、最外縁は `DIRT_PATH` で縁取り。
- **低い柵**: プレイフィールドを高さ1の `OAK_FENCE` で囲い、要所に `OAK_LOG` 柱＋`LANTERN`。壁ではなく牧場の柵。
- **立体的な建物・背景（横に配置）**: 大納屋（9×7・切妻屋根・木組み柱・窓・入口、高さ約9）、高いサイロ（円錐屋根・高さ約11）、給水塔、**回転翼の風車**、煙突付きコテージ、干し草の山、池、外周に背景の木立。
- **バッチ設置**: 1tickあたり `blocks-per-tick`（既定1800）個ずつ設置し、物理演算オフで連鎖更新を抑制。
- **破壊保護**: 生成物を 爆発 / 火・落雷着火 / 手壊し・エンダーマン / ピストン / 液体 から守る（アリーナ内部の作物は対象外）。
- **自動追従**: アリーナのサイズ/位置が変わると農場を自動で作り直す。
- **決定的なシーン**: 座標から同じ結果を再計算できるため、`clear` は「FarmMap が置いたブロックのみ」を正確に元へ戻す（後から置かれた物・地形は巻き込まない）。

## コマンド

`/farmmap <build|clear|level 1-5|reload|status>`（エイリアス `/fmap`、権限 `farmmap.use`=op）

| サブコマンド | 動作 |
|---|---|
| `build` | 農場マップ（土台＋建物・背景）を生成。既存があれば撤去してから作り直す。 |
| `clear` | FarmMap が置いたものを撤去。 |
| `level <1-5>` | 横の広がり（装飾余白）を 8/10/12/14/16 に切替えて再生成。 |
| `reload` | `config.yml` を再読み込み。 |
| `status` | 現在の状態・範囲・保護設定を表示。 |

## サイズの考え方（半径モデル）

farm の `arena.location` は**中心**、`sizeX/sizeZ` は**中心からの半径**。

- プレイフィールド = 中心 ± 半径（例 半径15 → 31×31）
- マップ外形 = 中心 ±（半径 + 余白）。**横(X)と奥(Z)で余白を別々**に設定可能。
  - 横幅 = 中心 ±（半径 + `map.margin-x`）／奥行き = 中心 ±（半径 + `map.margin-z`）
  - 例: 半径15・margin-x=56・margin-z=200 → **横143 × 奥431**（横2倍・奥10倍の長い農場）
- 高さの壁は無し（建物のみ高さを持つ）。建物はプレイ面のすぐ外に集約し、広い余白は中央の道と遠景の並木で奥行きを演出。

`/farmmap level 1-5` は横・奥を**同じ値に揃える**（対称化）クイック設定：

| レベル | 余白(横=奥) | マップ外形（半径15時） |
|---|---|---|
| Lv1 | 20 | 71×71 |
| Lv2 | 24 | 79×79 |
| Lv3 | 28 | 87×87 |
| Lv4 | 32 | 95×95 |
| Lv5 | 36 | 103×103 |

非対称（奥だけ長い等）にしたい場合は `config.yml` の `map.margin-x` / `map.margin-z` を直接指定して `/farmmap reload` → `/farmmap build`。

## 主な設定（config.yml）

| キー | 既定 | 説明 |
|---|---|---|
| `map.margin` | 20 | 余白の既定（margin-x/z 未指定時に使用） |
| `map.margin-x` | 56 | 横幅(X)方向の余白 |
| `map.margin-z` | 200 | 奥行き(Z)方向の余白 |
| `map.floor-y-offset` | 0 | 基礎床の Y（arena.y からのオフセット） |
| `map.fill-playfield-floor` | true | 内側に耕地の床を敷くか |
| `map.clear-above-floor` | false | 床の上1層を整地（配信外のみ推奨） |
| `build.blocks-per-tick` | 1800 | 1tickの設置上限（軽くするなら 800〜1000） |
| `build.apply-physics` | false | 物理演算（false=軽量・推奨） |
| `protect.*` | true | 爆発/火/破壊/ピストン/液体の保護 |
| `auto-follow.enabled` | true | アリーナ変化への自動追従 |
| `debug` | false | 設置数・所要tick等の詳細ログ |

## 導入・注意

- Paper（api-version 1.21 系。Farm 26.1.2 サーバーで動作確認想定）。
- 座標/サイズは `plugins/s2e-farm-pro/config.yml` を直読み（farm-pro 内部のリフレクションは不使用＝難読化に依存しない）。
- **同名プラグインの二重配置に注意**: `plugins/` には `FarmMap_v1.1.0.jar` だけを置く（旧 `FarmMap_v1.0.0.jar` は削除）。
- 位置がズレる場合は `arena.center-x-offset` / `center-z-offset` を調整して `/farmmap reload` → `/farmmap build`。
- コード変更の反映にはサーバー再起動が必要。作物が消える操作（`clear-above-floor` 等）は配信外で。

## 更新履歴

- **v1.3.0** — 余白を横(X)・奥(Z)で別々に設定可能に（`margin-x`/`margin-z`）。既定を 横56・奥200（横2倍・奥10倍の長い農場）に。建物をプレイ面のすぐ外へ集約し、広い余白を中央の十字路と遠景の並木・干し草で演出。
- **v1.2.0** — 余白の既定を 8→20 に拡大（横の広がりを大きく、level も 20/24/28/32/36）。建物を立体的に強化（大納屋・高いサイロ・給水塔・回転翼の風車・煙突付きコテージ）＋外周に背景の木立。
- **v1.1.0** — 横展開デザインへ転換。高さの壁を廃止し、平らな土台＋建物・背景（納屋・サイロ・干し草・池・かかし・木）を横に配置（スタイル #1 木の牧場）。`clear` をシーン再計算方式に（置いたものだけ正確に撤去）。`/farmmap` を素で実行可能に（全角エイリアス除去）。
- **v1.0.0** — 初版（枠ステージ）。基礎床＋外周境界10段の壁、バッチ設置、保護、自動追従。
